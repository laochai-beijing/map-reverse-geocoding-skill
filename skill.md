# 全球RGC逆地理编码服务 Skill 能力定义
## 1. 基础信息
- 技能名称：global-rgc-reverse-geocoding
- 技能版本：v1.0
- 适用智能体：云端智能体（Coze/Claude Code）、本地智能体（网易LobsterAI/OpenClaw）
- 核心定位：GIS/LBS高频逆地理编码通用能力，基于微博官方RGC接口实现全球覆盖与自动合规
- 开发适配：贴合微博接口原生输入输出，无额外二次封装，降低开发适配成本

## 2. 核心能力描述
- 核心目标：调用微博官方RGC接口，实现经纬度坐标到结构化地理信息的解析，同时完成**坐标自适应切换**与**国土政策自动合规**；
- 能力边界：支持全球经纬度（-180~180经度，-90~90纬度）解析，覆盖国内（含台湾、三沙）及海外所有区域，不支持涉密/军事禁区地理信息解析；
- 触发条件：智能体接收到「逆地理编码」「RGC解析」「坐标转地理信息」「全球位置解析」等指令时自动触发；
- 核心优势：替代高德地图仅国内的限制，实现全球一体化逆地理编码，合规与性能兼顾，适配GIS/LBS高频调用场景。

## 3. 输入参数规范
### 基础入参（与微博接口完全一致，无冗余）
| 参数名 | 类型   | 必选 | 取值范围       | 说明                                          |
|--------|--------|------|----------------|-----------------------------------------------|
| formpt | string | 是   | 自定义         | 微博RGC接口专属密钥，需私有化配置，禁止明文暴露 |
| lng    | float  | 是   | -180 ~ 180     | 待解析坐标经度，数字格式                      |
| lat    | float  | 是   | -90 ~ 90       | 待解析坐标纬度，数字格式                      |

### 入参约束
- 经纬度必须为纯数字/浮点型，禁止传入字符串、特殊符号；
- formpt密钥为必传项，缺失直接返回接口调用失败；
- 经纬度超出取值范围时，不发起接口调用，直接返回参数错误。

## 4. 输出参数规范
### 原生输出（完全贴合微博RGC接口返回结构）
| 一级字段 | 类型   | 说明                     | 二级核心字段       | 二级字段说明               |
|----------|--------|--------------------------|--------------------|----------------------------|
| status   | string | 接口调用状态             | -                  | ok=成功，其他=失败         |
| result   | object | 地理信息结果集           | 经纬度组合键       | 如113.552574_22.126112     |
| -        | -      | -                        | disctrict          | 区县级地理信息             |
| -        | -      | -                        | city               | 地市级地理信息             |
| -        | -      | -                        | province           | 省级地理信息               |
| -        | -      | -                        | country            | 国家级地理信息             |

### 地理信息子字段规范（所有层级通用）
| 子字段     | 类型   | 说明                     |
|------------|--------|--------------------------|
| objectid   | string | 地理信息唯一标识ID       |
| name       | string | 地理名称（标准全称）|
| simple_name| string | 地理名称（简称）|
| adcode     | string | 行政区划编码             |
| type       | int    | 地理层级类型（1=国家，2=省，3=市，4=区县） |
