# 全球RGC逆地理编码服务
GIS/LBS开发高频通用Skill，基于微博官方RGC接口实现全球逆地理编码，完美兼顾**全球覆盖、坐标无缝切换、国土政策自动合规**，替代高德仅支持国内坐标的局限性，接口响应高效适配高频调用场景。

## 核心特性
- 🌍 全球覆盖：支持国内+海外全区域逆地理编码，突破高德国内限制
- 📍 坐标自适应：国内自动适配WGS84坐标、海外自动切换GCJ02坐标，无缝衔接无需手动配置
- 📜 合规内置：接口层自动完成台湾、三沙群岛等国土敏感区域政策校验，无需用户额外选择
- ⚡ 高性能：基于微博官方RGC接口开发，响应速度满足GIS/LBS高频调用需求
- 🔑 密钥安全：联系chaiqi@staff.weibo.com 获得密钥使用（支持wb开发平台付费采购，价格远低于市场价格）

## 核心接口信息
- 接口地址：`https://place.weibo.cn/wandermap/rgc`
- 请求方式：GET
- 核心入参：formpt（密钥，私有化配置）、lng（经度）、lat（纬度）
- 输出格式：JSON（贴合微博接口原生返回结构，无额外冗余字段）

## 快速使用
### 入参规范
| 参数名 | 类型   | 必选 | 说明                     | 示例          |
|--------|--------|------|--------------------------|---------------|
| formpt | string | 是   | 接口专属密钥（需隐藏）| weibo-lbs |
| lng    | float  | 是   | 经度（范围：-180~180）| 113.552574    |
| lat    | float  | 是   | 纬度（范围：-90~90）| 22.126112     |
